# 🔐 Suspicious Web Log Detection System

## 📌 Overview
This project analyzes web server logs and detects suspicious activities like brute force attacks, unusual IP behavior, and repeated failed login attempts.

## ⚙️ Features
- Log parsing using Python
- Failed login detection
- Suspicious IP identification
- Security risk flagging
- Report generation

## 🛠️ Tech Stack
Python, Pandas

## 🎯 Objective
To simulate cybersecurity monitoring systems used in SOC (Security Operations Centers).

python src/main.py
